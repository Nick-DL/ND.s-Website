留言

本网站留言功能已于 2023 年 1 月 24 日上线。此功能目前基于 LeanCloud 网络服务平台和 Valine 评论管理系统。

此外，可以通过备用的 Gitalk 评论功能参与留言，需要 GitHub 账号，所以这里不推荐。参与 Gitalk 评论（没有启用） →

如果评论系统出现故障，可前往 GitHub Issues 页面留言。

小提示：评论时可以使用 Markdown 语法，实现加粗，斜体，引用，列出代码等等多种格式。具体说明 →

参加留言活动即代表您已阅读并认可本网站评论公约。
